### Контекст и ограничения
- **Канал:** только Telegram, без email/SMS
- **Кто пишет:** Ирина вручную в групповые чаты + опционально бот от имени Ирины (для личных приветствий)
- **Чаты:** два отдельных — чат предзаписи (free, все) и платный чат (только оплатившие)
- **Обращение:** «вы» везде
- **Тон:** голос Ирины — живой, с юмором, без давления и стыда

### Критерии готовности
1. Каждое сообщение содержит: дату/условие, канал (группа / личное), цель, полный текст
2. Обращение «вы» последовательно
3. Нет упоминания email/SMS
4. Продажные точки закрыты: открытие продаж, 4 ценовых волны, апсейл
5. Голос Ирины: живой, не «боты пишут»
6. Во время марафона: утреннее + вечернее сообщение для каждого дня (полный текст для ключевых дней 1/4/5/8/9/10; шаблон с подстановкой для 2/3/6/7)
7. Технические рекомендации по сервису для бота
8. Раздел «Открытые вопросы» в конце

### Хронология (относительная, без привязки к датам)
| Триггер | Событие |
|---|---|
| T-0 (старт предзаписи) | Чат предзаписи создан, Практика 1 в закрепе |
| ~T+3 дня | Практика 2 — публикуется Ириной как событие |
| ~T+5 дней | Открытие продаж. Цена 1 990 ₽ (волна 1, 3 дня) |
