# Целевая аудитория — актуальная версия

## Ядро: женщины 28–45

Сквозные характеристики ядра (общие для всех сегментов):
- Сидячая или малоподвижная работа (офис, удалёнка, фриланс)
- Живот — главная зона недовольства телом
- Паттерн «начинала и бросала»: пробовали тренировки / диеты, но не удержали
- Не готовы к жёстким диетам и интенсивному фитнесу
- Нужен понятный короткий формат, который встраивается в день

## Приоритетные сегменты

| Сегмент | Кто это | Боли | Что хочет получить | Главный триггер покупки |
|---|---|---|---|---|
| S1: Офис + вечерний живот | Женщины 28–45, сидячая работа, плотный график | Живот раздувается к вечеру, отёки, вздутие после еды, спина/шея от компьютера, усталость | Собранный живот, лёгкость к вечеру, понятный режим 15–20 мин | «Хочу быстрый ощутимый эффект без насилия» |
| S2: Стресс + нестабильность | Женщины 28–45, высокий стресс, недосып, нестабильный режим | Живот «не уходит» несмотря на попытки, стресс-еда, срывы, чувство потери контроля над телом | Устойчивая привычка, снижение стресс-живота, система «которую невозможно бросить» | «Хочу перестать срываться и наконец удержать результат» |
| S3: Дискомфорт + пищевое вздутие | Женщины 28–45, регулярный дискомфорт в ЖКТ | Вздутие после еды, не знает что вызывает дискомфорт, тяжесть, «рыхлость» в области живота | Понять триггеры вздутия, получить простые пищевые правила без диеты, визуально уменьшить живот | «Хочу разобраться, почему раздувает, и убрать это» |

**Пересечение сегментов:** В реальности участница чаще всего попадает в 2–3 сегмента одновременно. Основной профиль: «офисная работа + стресс + вздутие к вечеру». Сегментация нужна для рекламных углов, а не для разделения аудитории на группы.

## Возражения и отработка

| Возражение | Что за этим стоит | Как отвечаем (механика программы) |
|---|---|---|
| «У меня нет времени» | Страх длинных тренировок | 15–20 мин/день, `минимум дня` 5–10 мин, день 7 «экспресс за 10 мин» |
| «Я уже начинала и бросала» | Негативный опыт, самокритика | Антисрыв-блок (день 5), правило «плохая тренировка > никакая» (день 7), лайт/звёздочка варианты |
| «Онлайн не работает» | Скепсис к формату | 2 бесплатных дня с реальным телесным эффектом до оплаты |
| «Боюсь травм / не знаю технику» | Недоверие к онлайн | Прогрессии от простого к сложному, чек-лист техники (день 6), видео-фидбек (день 4) |
| «Не знаю, что вызывает вздутие» | Пищевая растерянность | Пищевой микро-трек: правило последнего приёма, трекер воды, правило 20 жеваний, памятка «5 триггеров + замены» |
| «Я в постоянном стрессе, мне не до тренировок» | Стресс как барьер | День 8 «стресс-живот»: вечерний восстановительный протокол, связь кортизол → живот, тест риска отката |

## Коммуникационные формулы

- «Живот — это не только жир. Отёк + вздутие + осанка визуально добавляют 2–4 см»
- «Убрать лишнее → включить своё → закрепить» (3 слоя программы)
- «Плохая тренировка лучше никакой»
- «Сначала почувствуете эффект телом — потом закрепите в режиме»
- «Не идеальность, а регулярность: лучше 10 минут, чем 0»
- «Результат через системность, а не через страдания»

## Рекламные углы по сегментам

| Сегмент | Угол для креатива | Обещание |
|---|---|---|
| S1: Офис + вечерний живот | «К вечеру живот как будто надувается? Вот почему — и как убрать за 15 минут» | Быстрый эффект: живот меньше уже на утро после дня 0 |
| S2: Стресс + нестабильность | «Стресс перераспределяет жир в живот. Как остановить это без диет» | Вечерний протокол, снижение кортизолового живота |
| S3: Дискомфорт + вздутие | «Вздутие после еды — это не норма. 4 простых правила без диеты» | Пищевой микро-трек + видимый эффект за 3 дня |
| Кросс-сегмент | «15 минут в день. Без диет. Без зала. Первые 2 дня бесплатно» | Формат + бесплатный вход |

## Дополнительные сегменты (кто ещё может прийти)

Не таргетируем активно, но принимаем, если придут. Используем для таргета, если приоритетные сегменты плохо отработают.

### D1: Предприниматели / руководители 30–45

| | |
|---|---|
| Кто | Женщины с высокой загрузкой, ненормированный график, мало времени на себя |
| Боли | Живот «вырос за последний год», перекусы на бегу, нет энергии на спорт, высокий кортизол |
| Желания | Вернуть контроль над телом без добавления ещё одного «проекта» в жизнь |
| Возражения | «У меня непредсказуемый график» → минимум дня 5 мин + день 7 «авральный». «Плачу за зал, но не хожу» → дома, без оборудования |
| Триггер | Системность + эффективность + минимальный формат |
| Где обитают | Telegram бизнес-каналы, Instagram бизнес-блогеры, подкасты, нетворкинг-чаты |
| [ГИПОТЕЗА] | Позиционирование «марафон для женщин» может считаться несерьёзным. Нужен креатив с акцентом на систему/эффективность. Доступа к этой аудитории нет, только платная реклама |

### D2: Женщины 45–52 (перименопауза)

| | |
|---|---|
| Кто | Женщины, у которых живот начал расти «сам по себе» при том же питании и активности |
| Боли | Гормональные изменения, задержка воды, перераспределение жира, чувство «тело стало чужим» |
| Желания | Понять, что происходит. Инструменты для «нового» тела. Мягкий подход без надрыва |
| Возражения | «Мне уже поздно» → начинаем с дыхания и мягких движений. «Это для молодых» → адаптации лайт/звёздочка |
| Триггер | Подход без стыда и без обещания чудес |
| Где обитают | Instagram lifestyle 40+, Telegram про здоровье, подкасты о женском здоровье, рекомендации подруг |
| [ГИПОТЕЗА] | Программа не адресует гормональный фактор прямо. День 8 (кортизол) частично покрывает, но перименопауза шире. Опыт Ирины с этим возрастом есть, но небольшой. Таргетируем, только если основные сегменты плохо отработают |

### D3: Женщины после диет (пост-диетный синдром)

| | |
|---|---|
| Кто | Пробовали кето / ИФ / калорийность — живот не ушёл или вернулся. ЖКТ расшатан |
| Боли | Вздутие, чувствительность к продуктам, «я всё перепробовала, ничего не работает» |
| Желания | Подход БЕЗ диеты. Разобраться с вздутием. Результат не через ограничения |
| Возражения | «Без диеты ничего не изменится» → живот — это не только жир. «Уже всё пробовала» → 2 бесплатных дня — попробуйте |
| Триггер | Позиционирование «БЕЗ диеты» — цепляет, потому что устали от диет |
| Где обитают | Instagram (питание/ПП/антидиеты), Telegram нутрициология, YouTube |
| [ГИПОТЕЗА] | Пищевой микро-трек — 4 точки, не нутрициология. Если придут с серьёзными ЖКТ-проблемами (СРК, непереносимости), программа это не закроет. В коммуникации: «привычки, а не лечение» |

### D4: «Тихие ЗОЖницы» (уже занимаются, но живот не уходит)

| | |
|---|---|
| Кто | Ходят на йогу / пилатес / бегают, но живот остаётся. Слабый глубокий кор при внешней активности |
| Боли | «Всё делаю правильно, а живот на месте». Не понимают, что упускают |
| Желания | Точечное решение именно для живота. Понять, почему не работает |
| Возражения | «У меня уже есть тренировки» → это дополнение: 15 мин на глубокий кор. «Нужно серьёзнее» → программа про глубокие мышцы, не классический пресс |
| Триггер | «Наконец объяснили, почему живот не уходит, хотя я занимаюсь» |
| Где обитают | Instagram фитнес/йога/пилатес, Telegram фитнес-каналы, оффлайн студии |
| [ГИПОТЕЗА] | Могут воспринять программу как слишком базовую. Но если приходят — потенциально хороший upsell-сегмент (нужна индивидуальная коррекция). Плохой для массового привлечения, хороший для конверсии в ведение |

## Профиль покупателя 8-недельного ведения (первое приближение)

На основе теста «готова ли идти одна» (день 9) и теста риска отката (день 8):

| Характеристика | Описание |
|---|---|
| Кто покупает | Участницы с 0–4 «да» в тесте дня 9 (не уверены, что удержат результат самостоятельно) |
| Типичный профиль | 30–42 года, высокий стресс, нестабильный режим, ранее бросала программы, 3+ баллов в тесте риска отката |
| Главная мотивация | Страх отката: «Я вижу результат, но знаю, что без поддержки потеряю» |
| Вторая мотивация | Незакрытые задачи: диастаз, индивидуальная коррекция техники, перестройка пищевого поведения |
| Что не работает | Давление и искусственный дефицит — работает честная диагностика ограничений (день 9) |
| Основной барьер | Цена: нужна ценностная рамка «сколько стоит 1 занятие с тренером офлайн vs 8 недель онлайн-ведения» |

## Анти-ЦА (кому НЕ подходит — не тратим бюджет)

| Кто | Почему не подходит | Как отсекаем |
|---|---|---|
| Люди, ожидающие «минус 10 кг за 2 недели» | Программа про визуальный эффект (осанка + отёк + кор), а не про похудение | Честные обещания на лендинге: «визуально минус 2–4 см, а не минус 10 кг» |
| Люди, которые хотят результат без действий | Нужно делать 15–20 мин/день минимум 8 из 10 дней | Отсекаются ценой + форматом ежедневных отчётов |
| Женщины после недавних родов без разрешения врача | Программа не содержит постпартум-адаптаций, нет работы с тазовым дном | FAQ на лендинге: «При диастазе или недавних родах — сначала к врачу» |
| Женщины с диагностированным диастазом | В марафоне нет диагностики и коррекции диастаза, день 9 прямо об этом говорит | FAQ + анти-ЦА блок «Для кого / не для кого» на лендинге |
| Люди с острыми состояниями (грыжи, обострения, послеоперационный период) | Нужен врач, а не онлайн-марафон | FAQ: «при острых состояниях — сначала к врачу» |
| Женщины с РПП (расстройство пищевого поведения) | Пищевой микро-трек может триггерить контроль/ограничения | FAQ: «если у вас сложные отношения с едой — проконсультируйтесь со специалистом» |
| Женщины с СРК / ГЭРБ / серьёзными проблемами ЖКТ | Программа — привычки, не лечение. Могут ожидать медицинский эффект | FAQ: «при хронических заболеваниях ЖКТ — сначала к гастроэнтерологу» |
| Мужчины | Программа и коммуникация — для женщин | Ясно из позиционирования и текстов |
| «Фитнес-туристки» (покупают всё, не делают ничего) | Даже антисрыв-механики не помогут без минимальной готовности | Сложно отсечь на этапе рекламы; частично отсекаются ценой |
| Женщины, ожидающие персональный формат за 2000 ₽ | Марафон — групповой, обратная связь ограничена | На лендинге: чётко описать формат (записи + чат + 2 эфира) |

## Гипотезы для проверки при запуске

| # | Гипотеза | Как проверить | Статус |
|---|---|---|---|
| H1 | D1 (предприниматели) реагируют на угол «система / эффективность», а не на «марафон для женщин» | Тестовый креатив с бизнес-тоном vs текущий | Ждёт запуска |
| H2 | D2 (45–52, перименопауза) приходят, но ожидают ответ на гормональный запрос, которого нет | Мониторить вопросы в чате в первом потоке | Ждёт запуска |
| H3 | D3 (после диет) — сильно цепляет «без диеты», но могут разочароваться, что пищевой трек — 4 точки | Мониторить отзывы + вопросы | Ждёт запуска |
| H4 | D4 («тихие ЗОЖницы») — плохой сегмент для привлечения, но хороший для upsell в ведение | Отслеживать: кто уже занимается → кто покупает ведение | Ждёт запуска |
| H5 | Угол «стресс-живот» (S2) даст CTR выше, чем «вечерний живот» (S1), потому что менее заезжен | A/B тест креативов | Ждёт запуска |
| H6 | Угол «вздутие» (S3) привлечёт часть людей с медицинскими проблемами, которых нужно отсеивать | Мониторить вопросы про СРК/непереносимости в чате | Ждёт запуска |
| H7 | Блок «для кого / не для кого» на лендинге повысит CR, а не снизит (эффект экспертности) | Замерить CR до/после добавления блока (или A/B) | Ждёт запуска |
