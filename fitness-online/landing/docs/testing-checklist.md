# Тест-чеклист лендинга

## Форма предзаписи
- [ ] Отправка с именем + согласием без телефона проходит успешно.
- [ ] Отправка без согласия блокируется.
- [ ] Некорректный телефон (если заполнен) показывает ошибку.
- [ ] После успешной отправки редирект на `/prechat-instruction`.

## UTM и lead_id
- [ ] UTM сохраняются при входе на `/`.
- [ ] UTM сохраняются в переходах `/ -> /pay -> /pay/success`.
- [ ] `lead_id` не теряется в цепочке страниц.

## Оплата и статусы
- [ ] На `/pay` корректно показана текущая цена.
- [ ] Таймер обновляется каждую секунду.
- [ ] При `?payment=failed` показывается уведомление об ошибке оплаты.
- [ ] В mock-режиме кнопка оплаты ведет на `/pay/success`.

## Цены и даты (МСК)
- [ ] На границе 13.03.2026 00:00 МСК цена переключается 1 990 -> 2 490.
- [ ] На границе 16.03.2026 00:00 МСК цена переключается 2 490 -> 2 990.
- [ ] На границе 19.03.2026 00:00 МСК цена переключается 2 990 -> 3 490.

## Мобильная версия
- [ ] На `/` и `/pay` главный CTA виден и кликабелен.
- [ ] Sticky CTA не перекрывает поля формы.
- [ ] На ширине 320px нет горизонтального скролла.
- [ ] Таймер и цена читаются без наложений.
