# D'Arky Dance — Структура и тексты лендинга (Этапы 2–3)

## Назначение
Один лендинг под холодный трафик для **парной бачаты с нуля**.
Цель страницы: конверсия в заявку на ближайшее пробное занятие.

## Рамки
- Структура и copy deck (этапы 2–3).
- Без кода/прототипа (этап 4), без дизайна (этап 5).
- Source of truth: `../brief.md`, `../offer-pair-bachata.md`, `landing-research.md`.

## Зафиксированные решения
1. Один LP под парную бачату для новичков.
2. Один URL, два варианта hero по UTM (`VK`, `Direct`).
3. Основной CTA: форма заявки.
4. Fallback: только Telegram.
5. Прайс-блок: только вход `800/400`.
6. Соцдоказательства: видео + короткие отзывы.
7. Обязателен короткий блок «Почему D’Arky».
8. Обязателен блок «2 ближайшие даты + пн/ср 21:00».
9. Лимит группы подаётся через заботу и персональное внимание, без цифры.
10. Юридика формы: согласие + placeholder ссылки на политику ПД.

## Каноническая структура страницы (10 блоков)

### BLK-01 | Hero (UTM-вариативный)
- **ID:** `BLK-01`
- **Название:** Первый экран
- **Цель:** За 5–10 секунд объяснить оффер и дать первый шаг к заявке.
- **Ключевая мысль:** «Парная бачата с нуля: можно без пары, без давления, с понятным входом».
- **Контент:** H1, подзаголовок, 3 буллета, живой визуал, основной CTA.
- **CTA:** `Записаться на пробное`.
- **Mobile-priority:** H1 + CTA + 2 буллета без перегруза; видео можно ниже кнопки.
- **Примечание для этапа 3:** Две версии текста по UTM.

- `Bullets_VK_1`: Без опыта и без своей пары — это нормальный вход.
- `Bullets_VK_2`: Пробное проходит в реальной группе, чтобы вы сразу поняли формат.
- `Bullets_VK_3`: В рабочее время отвечаем быстро и помогаем выбрать ближайший слот.
- `H1_Direct`: **Школа парной бачаты в Новосибирске: пробное 800 ₽ / 400 ₽**
- `Subhead_Direct`: Группа для новичков пн/ср в 21:00, Фрунзе 80. Можно начать с нуля и прийти без своей пары.
- `Bullets_Direct_1`: Чёткий формат для взрослых новичков.

## Контроль качества (этап 3)
1. Hero содержит 2 версии (`VK` и `Direct`), CTA одинаковый.
2. Во всех блоках текст ведёт к основной цели — заявке в форме.
3. Нет фейковой срочности и агрессивного давления.
4. Цена указана только как вход `800/400`.
5. Ограничение группы подано через заботу, без чисел.
6. Юридический блок формы включает согласие и placeholder-ссылку на политику ПД.
7. Lady Style не включён в scope этого лендинга.
