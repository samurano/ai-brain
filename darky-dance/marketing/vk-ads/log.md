## 2026-03-05 (Composio usage script за текущий месяц)
- Добавлен CLI-скрипт:
  - `marketing/vk-ads/scripts/composio_month_usage.py`
- Скрипт считает usage через:
  - `POST https://backend.composio.dev/api/v3/internal/action_execution/logs`
  - авторизация: `x-api-key`.
- Реализовано:
  - период `from/to` в epoch ms (с 1-го числа месяца по текущий момент),
  - пагинация `cursor/nextCursor`,
  - итоговые метрики:
    - `used_calls`
    - `remaining_calls` (из `20000`).
- Для premium вызовов включён честный fallback:
  - если из логов нельзя надёжно выделить premium для всех записей,
  - скрипт выводит `used_premium_calls: unavailable`.
- Фикс после первого запуска:
  - добавлены `Accept/User-Agent` заголовки,
  - скорректирован тип `cursor` (число, не строка),
  - добавлен парсинг ответа, где `data` приходит списком и `nextCursor` в корне.
