# D'Arky Dance — VK Ads Tests Log

## Как вести журнал
1. Один тест = одна отдельная карточка.
2. Ведём файл append-only (новые тесты только в конец).
3. Обязательные поля не пропускаем: аудитория, текст, визуал, плейсмент, бюджет, период.
4. Результаты заполняем только по данным выгрузки из `data/`.
5. В блоке «Решение» выбираем один основной статус и при необходимости дописываем конкретные правки.

## Правило нумерации
- Номер теста: `#001`, `#002`, `#003`...
- Если тест перезапускается после правок, используем суффикс версии: `#003-v2`.
