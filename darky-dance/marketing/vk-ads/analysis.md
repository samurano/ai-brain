# D'Arky Dance — VK Ads Analysis

## Как обновлять этот файл
- Обновление после каждой новой выгрузки из `data/` (минимум 1 раз в неделю).
- Каждая неделя = новая секция в конце файла.
- В выводах опираемся на факты из выгрузок и карточек `tests-log.md`.
